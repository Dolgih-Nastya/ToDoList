== README

В проекте ToDoList работают такие функции
* Удаление проекта
* Удаление таска
* Редактирование проекта
* Редактирование таска
* Возможность отметить таск как сделанный
В проекте ToDoList следующие функции работают только после перезагрузки страницы:
* Добавление проекта
* Добавление таска
Другие
* Опеределение приоритета, но без сохранения
